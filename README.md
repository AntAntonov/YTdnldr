# YTdnldr

A simple Python script with a GUI for downloading MP3s.

## Features
- Download MP3 files
- Graphical User Interface (Tkinter)
- Select download folder
- Lightweight and efficient

## Installation
```sh

# Import dependencies
os
tkinter
yt_dlp 
```

## Usage
```sh
Just run in terminal
```
